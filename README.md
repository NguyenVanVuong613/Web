# B2: Tải NPM package cho cả 2 folder BE và FE, mở terminal của cả 2 Folder và dùng lệnh:
```cmd
npm install
```
# B3: Sau khi đợi tải xong ae dùng lệnh dưới để chạy (chạy lệnh trong Folder BE trước sau đó chạy lệnh trong Folder FE):
```cmd
npm start
```
